#ExtendedUsage
---++ Extended Usage

There are several other ways to put an image in a topic:

---+++ ICON Variable

Use any of the doc graphics defined in %CALCULATE{$GET(icontopics)}%. The ICON variable supports these parameters:

| *Parameter:* | *Description:* | *Default:* |
| ="name"= | Name of icon | required |
| =format="..."= | Format of icon. Supported variables (using %ICON{person}% =%<nop>ICON{"person"}%= as example): %BB% =$name= - name of icon (=person=) %BB% =$type= - type of icon (=gif=) %BB% =$filename= - icon filename (=person.gif=) %BB% =$web= - web where icon is defined (=TWiki=) %BB% =$topic= - topic where icon is defined (=TWikiDocGraphics=) %BB% =$description= - icon description (=Person=) %BB% =$width= - width of icon (=16=) %BB% =$height= - height of icon (=16=) %BB% =$img= - full img tag of icon (=&lt;img src="... /&gt;=) %BB% =$info= - img tag with usage info in title (=&lt;img src="... /&gt;=) %BB% =$url= - URL of icon (=http://example.com/pub/TWiki/TWikiDocGraphics/person.gif=) %BB% =$urlpath= - URL path of icon (=/pub/TWiki/TWikiDocGraphics/person.gif=) | =format="$img"= |
| =default="else"= | Alternate icon if named icon is not defined | =default="$name"= |

*Note:* For the icon name, only the file type portion is used if a full URL or file path is specified. For example, a pdf icon is shown for icon name =/home/sweet/home.pdf=.

*Examples:*

   * =%<nop>ICON{"flag-gray"}%= returns: %ICON{"flag-gray"}%
   * =%<nop>ICON{"pdf"}%= returns: %ICON{"pdf"}%
   * =%<nop>ICON{"smile.pdf"}%= returns: %ICON{"smile.pdf"}%
   * =%<nop>ICON{"/dont/you/dare/smile.pdf"}%= returns: %ICON{"/dont/you/dare/smile.pdf"}%
   * =%<nop>ICON{"http://twiki.org/doc/xhtml.xsl"}%= returns: %ICON{"http://twiki.org/doc/xhtml.xsl"}%
   * =%<nop>ICON{"bubble" format="$description icon is defined in [<nop>[$web.$topic]]"}%= returns: %ICON{"bubble" format="$description icon is defined in [[$web.$topic]]"}%

---+++ ICONURL Variable

Whereas the ICON variable returns an HTML img tag by default, the ICONURL variable returns just the URL of a doc graphic image. For example, =%<nop>ICONURL{"thumbs-up"}%= is a shortcut for =%<nop>ICON{"thumbs-up" format="$url"}%=, returning =%ICONURL{"thumbs-up"}%=, which renders as %ICONURL{"thumbs-up"}%.

---+++ ICONURLPATH Variable

The ICONURLPATH variable returns just the URL path of a doc graphic image. For example, =%<nop>ICONURLPATH{"thumbs-up"}%= is a shortcut for =%<nop>ICON{"thumbs-up" format="$urlpath"}%=, rendering as =%ICONURLPATH{"thumbs-up"}%=.

---+++ Icon shortcuts with preferences settings

In [[%USERSWEB%.TWikiPreferences]] set a variable to an ICON, for example:

=<nop>   * Set H = %<nop>ICON{help}%=

Now you can use the icon by writing =%<nop>H%=

---+++ Emoticons in !SmiliesPlugin

The [[SmiliesPlugin]] uses a different shorthand than =<nop>%VARIABLES%=. For example, =:-)= renders as :-)

---+++ Links with icons

To create a link with an image, write:%BR%
=[<nop>[%HOMETOPIC%][%<nop>ICON{home}% Home]]= to get: [[%HOMETOPIC%][%ICON{home}% Home]]

To get rid of the underline under the space, write: %BR%
=[<nop>[%HOMETOPIC%][%<nop>ICON{home}%]] [<nop>[%HOMETOPIC%][Home]]= to get: [[%HOMETOPIC%][%ICON{home}%]] [[%HOMETOPIC%][Home]]

---+++ List all icons

| *Format:* | *List all icons,%BR% including aliases:* | *List unique icons only:* | *Example:* |
| *Name list*, %BR%comma-space delimited: | =%<nop>ICON{list:all-names}%= | =%<nop>ICON{list:unique-names}%= | addon, air, airplane, ... |
| *Icon list*: | =%<nop>ICON{list:all-icons}%= | =%<nop>ICON{list:unique-icons}%= | %ICON{addon}% %ICON{air}% %ICON{airplane}%... |
| *Info list*, %BR%icons with usage info: | =%<nop>ICON{list:all-info}%= | =%<nop>ICON{list:unique-info}%= | %ICON{"addon" format="$info"}% %ICON{"air" format="$info"}% %ICON{"airplane" format="$info"}%... |
| *Detailed list*, %BR%table format: | =%<nop>ICON{list:all-table}%= | =%<nop>ICON{list:unique-table}%= | (similar to table format above) |

%T% __Hint:__ The =$LIST...()= functions of the SpreadSheetPlugin can be used to manipulate a name list. Example:
<verbatim>
%CALCULATE{$LISTJOIN($sp, $LISTEACH($NOP(%)ICON{$item}$NOP(%), %ICON{list:unique-names}%))}%
</verbatim>
Is equivalent to:
<verbatim>
%ICON{list:unique-icons}%
</verbatim>

---++ How to add doc graphic images

Additional doc graphic images can be defined. Do not add them to this topic because the modifications will be lost on the next TWiki upgrade. Instead add them to %CALCULATE{$IF($EXISTS(%USERSWEB%.SiteDocGraphics), $SET(usericontopic, [[%USERSWEB%.SiteDocGraphics]]), $SET(usericontopic, [[%SCRIPTURL{edit}%/%USERSWEB%/SiteDocGraphics?templatetopic=%SYSTEMWEB%.TWikiDocGraphicsTemplate;topicparent=none][%USERSWEB%.SiteDocGraphics]]))$GET(usericontopic)}%.

__Note:__ Documentation graphics are defined in %CALCULATE{$GET(icontopics)}%. The list of documentation graphics topics is defined by the ICONTOPIC setting in [[%SYSTEMWEB%.TWikiPreferences]] and can be overloaded in [[%LOCALSITEPREFS%]].

#RefreshIconCache
---+++ Refresh icon cache

TWiki caches icons for performance. The cache gets updated automatically when you update !TWikiDocGraphics topics or !TWikiPreferences topics. You can force a cache refresh with =%<nop>ICON{action:refresh-cache}%=.

%IF{ "$'URLPARAM{icon-cache}'='refresh'" then="$percntICON{\"action:refresh-cache\" format=\"ICON cache is now refreshed. <a href='%SCRIPTURL{view}%/%WEB%/%TOPIC%#RefreshIconCache' class='twikiButton'>OK</a>\"}$percnt" else="<a href='%SCRIPTURL{view}%/%WEB%/%TOPIC%?icon-cache=refresh#RefreshIconCache' class='twikiButton'>Refresh ICON cache now</a>" }%
